Utilities

Utilities provide small, reusable classes to control spacing, typography, colors, borders, display, and more. Oliver CSS includes a large set of fast and flexible utility classes.

Spacing

Use .m-* and .p-* for margin and padding.

margin 4
padding 4
<div class="m-4">Margin 4</div>
<div class="p-4">Padding 4</div>
