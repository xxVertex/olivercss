Getting Started

This guide shows you how to install and use the Oliver CSS Framework in your projects. Choose the method that best fits your workflow.

1. Download or Include the Framework

You can use Oliver CSS by downloading the compiled files or linking directly in your HTML.

CDN (coming after NPM publish)

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/cssoliver/dist/oliver.min.css">

Local File

<link rel="stylesheet" href="dist/oliver.min.css">

2. Install with NPM

If you're using a build system, install Oliver CSS as a dependency:

npm install cssoliver

Import it in your CSS:

@import "cssoliver/dist/oliver.css";

3. Framework File Structure

Oliver CSS is built from modular files separated into core utilities and UI components:

src/
  reset.css
  variables.css
  typography.css
  layout.css
  animations.css
  responsive.css
  utilities.css

  components/
    buttons.css
    cards.css
    alerts.css
    navbar.css
    modals.css
    forms.css

  framework.css    (imports everything)

The build process compiles these into:

dist/
  oliver.css
  oliver.min.css

4. Simple Usage Example

After installing the framework, you can immediately use its classes:

<button class="btn btn-primary">Primary Button</button>

<div class="card p-4 rounded shadow">
  <h3>Welcome</h3>
  <p>This card is styled with Oliver CSS.</p>
</div>

5. Rebuilding the Framework (Developers)

If you want to modify the source files, rebuild the framework using:

npm run build

This will regenerate both:

  • ✔ dist/oliver.css
  • ✔ dist/oliver.min.css
