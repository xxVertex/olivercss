Components

Oliver CSS includes carefully crafted UI components that are easy to use and fully customizable. Below are examples and usage code for each component.

Buttons

Use button variants such as primary, secondary, danger, outline, and more.

<button class="btn btn-primary">Primary</button>
<button class="btn btn-secondary">Secondary</button>
<button class="btn btn-danger">Danger</button>
<button class="btn btn-outline">Outline</button>

Cards

Cards include padding, shadows, and optional headers/footers.

Card Title

This is a simple card built with Oliver CSS.

<div class="card p-4 shadow rounded">
  <h3>Card Title</h3>
  <p>This is a simple card built with Oliver CSS.</p>
</div>

Alerts

Use alerts to display important messages or warnings.

Success! Everything looks good.
Warning! Check your input.
Error! Something went wrong.
<div class="alert alert-success">Success message</div>
<div class="alert alert-warning">Warning message</div>
<div class="alert alert-danger">Error message</div>

Navbar

A clean, flexible navigation bar.

OliverCSS
  • Home
  • Docs
  • GitHub
<div class="navbar">
  <div class="navbar-brand">OliverCSS</div>
  <ul class="navbar-nav">
    <li><a href="#">Home</a></li>
    <li><a href="#">Docs</a></li>
    <li><a href="#">GitHub</a></li>
  </ul>
</div>

Modals

Modals are centered containers used to display dialogs.

Modal Title

This is a modal example.

<div class="modal">
  <div class="modal-content p-4 rounded">
    <h3>Modal Title</h3>
    <p>Modal text...</p>
  </div>
</div>

Forms

Form elements styled with spacing, borders, and focus states.

<label>Name</label>
<input class="form-input" type="text" placeholder="Enter name">

<label>Email</label>
<input class="form-input" type="email" placeholder="Enter email">
