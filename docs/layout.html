Layout

Oliver CSS includes a lightweight grid system, containers, and responsive helpers. Use layout utilities to structure your content quickly and consistently.

Container

Use .container for centered content with a max width.

This is a container. 
<div class="container">
  Your content...
</div>

Grid System

The grid uses .row and .col (or numbered variants like .col-6).

.col
.col
<div class="row">
  <div class="col">.col</div>
  <div class="col">.col</div>
</div>

Fixed Column Sizes

.col-4
.col-8
<div class="row">
  <div class="col-4">.col-4</div>
  <div class="col-8">.col-8</div>
</div>

Responsive Columns

Use .col-md-* and .col-lg-* for responsive behavior.

Responsive
Responsive
Responsive
<div class="row">
  <div class="col-12 col-md-6 col-lg-4">Column</div>
  <div class="col-12 col-md-6 col-lg-4">Column</div>
  <div class="col-12 col-md-6 col-lg-4">Column</div>
</div>
  • .col-12 – full width on small screens
  • .col-md-6 – half width on medium screens
  • .col-lg-4 – one-third width on large screens

Flex Layout Helpers

</div>
